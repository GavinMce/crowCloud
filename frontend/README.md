# crowCloud frontend

React + TypeScript, built with Vite. See [Frontend Development](../CONTRIBUTING.md#frontend-development)
in the root `CONTRIBUTING.md` for the directory structure, the
`EntityLayout`/hub architecture, and the UI conventions this app follows.

```bash
npm ci
npm run dev          # dev server on :5173, proxies /api to :8080
npm run type-check
npm run lint
npm run test
npm run build
```
